# Skeumorphix

A skeumorphic experminational gui inspired by reapoer and [colorpalette.pro](https://colorpalette.pro/) for a unity/godot like app.

## Resources
* Wood texture from https://unblast.com/5-free-seamless-wood-textures-jpg/
* World editor is a cropped trenchbroom screenshot from https://trenchbroom.github.io/manual/latest/
* Model/particle editor is a cropped blender screenshot from https://docs.blender.org/manual/en/latest/editors/3dview/startup_scene.html
